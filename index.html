ProGaming046_pbp UI Mod Installation Guide

Follow these steps to install the UI Mod used by ProGaming046_pbp for Diablo 2 Resurrected.

  1. Download the Mod Files.

    Get the mod files here and save them to your PC.

  2. Extract the RAR.

    Extract the contents into a folder of your choice.

  3. Locate Your Game Directory.

    Find where Diablo 2 Resurrected is installed on your system.

  4. Create the Mods Folder.

    This step is only required if the folder doesn't already exist. Name it mods (all lowercase) and place it in the game's root directory alongside the executables.

  5. Move Mod Files.

    Drag and drop the danimal folder into the mods folder.

  6. Set Up Launch Parameters.

    In the Battle.net Launcher, click the gear on Diablo II Resurrected, open Game Settings, enable “Additional command line arguments,” and paste:

    -mod danimal -txt
  7. You're Done!

    The mod should now load automatically whenever you start the game.
