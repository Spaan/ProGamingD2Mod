Mod Installation Guide

Follow these steps to install the UI Mod used by ProGaming046_pbp for Diablo 2 Resurrected.

  1. Download the Mod Files.

    Go to the release page and download the latest ZIP archive.

  2. Extract the ZIP.

    Unzip the contents into a folder of your choice.

  3. Locate Your Game Directory.

    Find where Diablo 2 Resurrected is installed on your system.

  4. Copy Mod Files.

    Drag and drop all files from the extracted folder into your game directory.

  5. Launch the Game.

    Start Diablo 2 Resurrected as usual. The mod should load automatically.

    // Example launch command on Windows
"C:\Program Files\Diablo II Resurrected\D2R.exe" --txt
